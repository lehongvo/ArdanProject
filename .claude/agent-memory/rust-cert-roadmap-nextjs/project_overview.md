A production Next.js web app for tracking a 12-month learning journey: Rust → Solana → ZK Proofs → Web3 job.

**Why:** The user wants to pass the Ardan Labs Rust Certification and land a remote Web3/Systems job.

**10 Phases:**
1. Rust Foundations (Weeks 1-4)
2. Rust Advanced (Weeks 5-8)
3. Ardan Labs Certification Prep (Weeks 9-12)
4. LeetCode Grind 1000 (Weeks 13-16)
5. Solana Basics (Weeks 17-20)
6. Anchor Framework (Weeks 21-28)
7. DeFi & Advanced Solana (Weeks 29-32)
8. Blockchain Systems (Weeks 33-36)
9. ZK Proofs (Weeks 37-44)
10. Job Hunt & Interview Prep (Weeks 45-48)

**Pages:**
- `/` — Dashboard with TodayCard, StreakCounter, PhaseProgress, QuickStats
- `/roadmap` — 10-phase overview grid + timeline
- `/phase/[id]` — Phase detail with week accordion + task checklist
- `/progress` — Charts (LineChart, RadarChart), heatmap, mock exam history
- `/mock-exam` — Full exam simulator with timer, 50+ questions, results
- `/leetcode` — 1000-problem tracker with filters
- `/projects` — 10 Solana project cards with status tracking
- `/job-tracker` — Kanban with dnd-kit drag/drop
- `/resources` — Curated links by phase

**How to apply:** The project root is /Users/user/Desktop/ArdanProject (NOT in a subfolder).
